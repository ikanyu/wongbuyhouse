# WongBuyHouse
**This repository is no longer maintained since last commit.**

App to summarize and monitor pricing of selected properties my family is aiming to purchase

## Setup
```
git clone https://github.com/ikanyu/wongbuyhouse.git
cd wongbuyhouse
bundle install
```

## Setup Database
```
rake db:setup
```

## Start Server
```
rails s
```
